##  Objectif du projet
Ce mini-projet a pour objectif de consolider les bases de l’apprentissage supervisé à travers l’implémentation et l’analyse de deux modèles fondamentaux :
- la **régression linéaire** (MLS),
- la **régression logistique** (MLS).

---

##  Partie 1 – Régression Linéaire (MLS)
- Dataset utilisé : *Medical Insurance Cost* (Kaggle)
- Variable cible numérique : `charges`
- Étapes suivies :
  1. Analyse exploratoire des données
  2. Étude des corrélations
  3. Prétraitement des variables (One-Hot Encoding)
  4. Modélisation par régression linéaire
  5. Évaluation avec MSE et R²
  6. Interprétation des coefficients du modèle

---

##  Partie 2 – Régression Logistique (MLS)
- Dataset utilisé : *Iris* (scikit-learn)
- Problème transformé en classification binaire (Setosa vs autres)
- Étapes suivies :
  1. Préparation et normalisation des données
  2. Modélisation par régression logistique (fonction sigmoïde)
  3. Évaluation via matrice de confusion
  4. Calcul des métriques : Accuracy, Précision, Recall
